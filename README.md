# Neural-Networks---Deep-Learning-2
ML models for image classification tasks.

The assignment consists of four parts. The first part is about using the k-NN and NCC algorithms for classification of the CIFAR-10 dataset. The second uses a fine-tuned CNN for the same goal. The third part follows up with the use of an SVM. For the fourth part, we abandod CIFAR-10 and work on MNIST. We train Autoencoders on digit reconstruction and next digit reconstruction. 

For more details you can inspect the attached PDF files that contain the Report for each part.
